CSS selectors and specificity

Syntax Name Description
body {}; Tag Selector Use for selecting HTML tags
.class {}; Class Selector Use for selecting classes. Remember classes should / can be used in more than one place
#id {}; ID Selector Use for selecting specific IDs. Remember IDs can be used only for specific element.
h1, h2 {}; Groups of CSS Selectors Match multiple selectors to the same CSS rule, using a comma-separated list.
... !important; !important Rule Used on declarations to override any other declarations for a property and ignore selector specificity.
tag.class {} Chaining Selectors CSS selectors can be chained so that rule sets apply only to elements that match all criteria.
